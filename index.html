Wer sind wir?

Æonforge Empire ist ein Mittelalterlicher Minecraft Rollenspiel Server auf welchem du als Spieler die Möglichkeit hast deinen Wunschcharakter ausspielen zu können. Egal ob Mensch, Elf, Zwerg, Ork oder Mischwesen - Hier kannst du dich Frei Entfalten und dein Rollenspiel ganz nach deinen Wünschen Gestalten!

Rollenspiel

Erschaffe deinen Charakter nach deinen Vorstellungen! Egal ob einfacher Schmied, Talentierter Koch oder Großer Herrscher über Königreiche - Hier entscheidest du allein deinen Weg!

  • RP Events
  • Spannende Lore
  • Verschiedene Rassen
  • Ausgefallene Berufe

RPG

Erlebe Einzigartige RPG Elemente die das Spielen zu einem Unvergesslichen Erlebnis machen. Erledige Aufgaben, Erkunde Gebiete, Verteidige Städte

  • Quests
  • Spezielle Gebiete
  • Verschiedene Custom Items
  • Neue Rezepte

Warum Voten?

Ein Vote hilft dem Server eine größere Reichweite zu generieren um noch mehr Spieler auf Æonforge Empire zu bringen. Durch deinen Vote steigt der Server in der Rangliste und du bekommst sogar noch eine Nette belohnung!

Server FAQs.

Hier findest du Häufig gestellte Fragen und die dazu passenden Antworten

Auf Welcher Version läuft der Server?
Der Server läuft Aktuell auf der Minecraft Version 1.21.4 und ist nur mit dieser Erreichbar.
Wie Starte ich auf dem Server?
Eine Ausführliche Anleitung für Neulinge findest du im Forum unter https://aeonforge-network.net
Gibt es PvP auf dem Server?
Jein - Auf dem Server ist PvP Erlaubt, jedoch nur wenn der gegenüber eingewilligt hat, es einen RPlichen Angriffsgrund gibt oder die Person einer Kriegspartei angehört.
